# Alka Web

这是一个基于 Next.js 15 和 React 19 的现代化 Web 应用项目。

## 环境要求

- Node.js >= 22.6.0
- pnpm >= 10.5.2 (通过 Corepack 管理)

## 开始使用

### 启用 Corepack

Corepack 是 Node.js 官方的包管理器管理工具，用于确保团队使用统一的包管理器版本。

```bash
# 启用 Corepack
corepack enable

# 确认 pnpm 可用
pnpm --version
```

### 安装依赖

```bash
pnpm install
```

### 开发

```bash
pnpm dev
```
